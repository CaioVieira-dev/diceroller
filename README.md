## Tecnologias utilizadas
- ReactJS
- ContextAPI
- Typescript
- SASS

### Como instalar (React)

* Para baixar o projeto siga as instruções abaixo:

```
1. git clone https://github.com/CaioVieira-dev/diceroller.git
2. cd diceroller
```

* Instale as dependências do projeto:

```
3. yarn install
4. yarn start
```

ou

```
3. npm install
4. npm start
```
